## 프로그램설명
> 이프로그램은 영어 해석이 익숙지않은 분들위해 재작되었습니다.
> 이프로그램은 KAKAO API를 이용한 변역 프로그램입니다.

## 환경
> PYTHON 3.8.12 이상
> 
> 모든 os 에서 동작 가능
> 
> linux gcc 7.5.0 이상

## 라이브러리  설치
> ```
> installer.py
> ```
## 파일 설명
> main.py 주된 프로그램 파일
> installer.py 라이브러리 파일
## 실행방법
>```
>$ python installer.py
>$ python main.py
>```
